# Creatura Portfolio

クリエイター向けワークフローと制作ログ管理をテーマにしたポートフォリオリポジトリです。  
今後増やしていく複数言語のデモと、Web ポートフォリオサイトをひとつのリポジトリで管理します。

- `web` … Web ポートフォリオ（TypeScript 想定）
- `demos` … 各種デモコード（現在は Kotlin のみ、今後 Python / C# など追加予定）
