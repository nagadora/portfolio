EnvSnap – 問い合わせ用環境情報ヘルパー

OS / CPU / メモリなどの環境情報を自動で取得し、 Live2D Cubism などのお問い合わせフォームにそのまま貼り付けられる形で 出力する小さなユーティリティです。

C# / Kotlin / Python Windows / macOS

Overview

Live2D Cubism のお問い合わせフォームでは、OS名・PC機種名・CPU / GPU 型番・ メモリ容量など、多くの環境情報を手入力する必要があります。 EnvSnap は「毎回これを調べて書くのは大変」という実感から生まれたツールで、 ワンクリックで環境情報を取得し、テキストとしてコピーできるようにすることを 目的としています。

Tech & Role

Python コア

OS / CPU / メモリ / GPU などの情報取得と整形ロジックを Python で実装し、 Windows / macOS 共通の「コア」として扱います。

C# / Kotlin ラッパー

Windows 向けに C#、macOS / JVM 向けに Kotlin を用いて、 ボタン一つで実行・コピーができる軽い UI / CLI ラッパーを用意する構成を 想定しています。

Motivation

実際にお問い合わせフォームを確認した際、 「不具合を共有したいだけなのに、環境情報をまとめるだけで手間がかかる」 と感じたことがきっかけです。 単に不満として指摘するのではなく、 自分自身や他のクリエイターが少しでも楽になるツールとして形にしています。
