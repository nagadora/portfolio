Art Guard Lab – 画像保護フィルタ実験ツール

人の目には綺麗に見えることを大事にしながら、生成系 AI に学習させにくくする画像変換を試すための実験ツールです。

Role: 個人開発 Tech: TypeScript / Kotlin / Python / C# / Unity

Overview

画像に対してウォーターマークやノイズ付与など複数の処理を組み合わせ、 生成系 AI による特徴抽出を難しくしつつ、人間が見るぶんには違和感を抑えた出力を目指すプロジェクトです。 「完全な防御」ではなく、クリエイターが取れる選択肢を少しでも増やしたいという思いからスタートしています。

Architecture & Tech Stack

TypeScript（Web / ツール UI）

Before / After の比較ビューや、フィルタ設定のプレビュー UI を TypeScript で実装。 手元でパラメータをいじりながら結果の雰囲気を確認できる Web ツールを想定しています。

ソースコード： code/art-guard-lab/ts/

Kotlin（CLI / 軽いサーバー）

どのフィルタをどの順番で適用するか、パラメータをどう組み合わせるかといったパイプライン定義や設定管理を Kotlin で記述。 CLI ツールや軽量 API サーバーとして動作させる想定です。

ソースコード： code/art-guard-lab/kotlin/

Python（画像処理 / AI 評価）

実際の画像処理は Python ライブラリ（Pillow / OpenCV など）で実装し、 バッチ処理でまとめて変換。将来的には AI モデルを用いた効果検証や自動評価も Python 側で行う予定です。

ソースコード： code/art-guard-lab/python/

C# / Unity（Live2D 周辺）

変換済みテクスチャを Live2D や 2D/3D モデルに貼ったときの見え方を確認するため、 Unity 上にビューアシーンを用意し、C# スクリプトから操作できる構成を想定しています。

ソースコード（構想・メモ含む）： code/art-guard-lab/unity-csharp/

Code Layout

このプロジェクトのコードは、ポートフォリオリポジトリ内で次のように配置しています。

TypeScript（Web / ツール）：
code/art-guard-lab/ts/
Kotlin（CLI / 軽いサーバー）：
code/art-guard-lab/kotlin/
Python（AI・画像処理）：
code/art-guard-lab/python/
C# / Unity（Live2D 周辺）：
code/art-guard-lab/unity-csharp/
